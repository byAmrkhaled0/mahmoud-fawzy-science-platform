# ترقية المنصة إلى V56

هذا الإصدار يحتفظ بكل محتوى المنصة ووظائفها، ويعيد تنظيم واجهة الطلاب داخل لوحة الإدارة لتكون أسرع وأقل ازدحامًا.

## قبل النشر

1. احتفظ بنسخة احتياطية من بيانات المنصة.
2. نفّذ `npm test` ثم `npm run build`.
3. انشر قواعد التخزين والدوال أولًا، ثم ملفات الموقع؛ لأن الإصدار يوحّد أكواد الدخول ويحسّن التحقق من ملفات الواجبات.

```powershell
npm run firebase:deploy:backend
npm run firebase:hosting
```

أو استخدم الأمر الكامل:

```powershell
npm run firebase:deploy:all
```

## ما تغيّر في صفحة الطلاب

- كل طالب يظهر في صف مدمج على الكمبيوتر وكارت على الهاتف.
- الحالة والحضور ومتوسط الدرجات ظاهرون بسرعة.
- التعديل والملف هما الإجراءان الأساسيان الظاهران.
- الحضور وواتساب ونسخ الأكواد وتغييرها والحذف ما زالت موجودة داخل قائمة «إجراءات».
- الحذف وتغيير الأكواد يظلان متاحين للمدرس أو المدير فقط.

## ملاحظات التوافق

- الأكواد القديمة ما زالت تعمل، ويمكن استخدام زر «ترقية الأكواد القديمة» لتحويلها تدريجيًا.
- الأكواد الجديدة للطالب وولي الأمر تتكون من 8 أرقام.
- الملفات المسموح بها للواجب: JPG وPNG وWebP وPDF، بحد أقصى 10 ميجابايت.
