## التشغيل المحلي

```powershell
npm install
npm run dev
```

ثم افتح:

```text
http://127.0.0.1:5173
```

## الاختبار والبناء

```powershell
npm test
npm run build
```

نسخة النشر تُنشأ داخل `dist/`.

## النشر على Firebase

لنشر القواعد والدوال والاستضافة مرة واحدة:

```powershell
npm run firebase:deploy:all
```

أو على مرحلتين:

```powershell
npm run firebase:deploy:backend
npm run firebase:hosting
```

## متطلبات مهمة

- تثبيت المنصة على الهاتف يتطلب تشغيل الموقع عبر HTTPS.
- النسخ الاحتياطي المجدول يعتمد على Cloud Scheduler، وقد يتطلب خطة Firebase تدعم Cloud Functions المجدولة.
- تفعيل المصادقة الثنائية لحسابات فريق العمل يتم من إعدادات Firebase Authentication / Identity Platform.
- راجع `UPGRADE_V56_AR.md` قبل النشر، خصوصًا ترتيب نشر الدوال ثم الاستضافة.
